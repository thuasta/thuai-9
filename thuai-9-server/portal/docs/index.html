比赛平台 公开观战台

Quick Start

参赛流程

  1. 在比赛平台注册队伍或登录已有队伍账号。
  2. 准备代码压缩包，压缩包根目录必须包含 Dockerfile。
  3. 在“提交代码”区域上传 .zip 文件，单个文件最大 10 MB。
  4. 等待评测 worker 编译并安排对局，结果会进入排行榜。
  5. 通过公开观战台查看实时游戏状态；观战台默认连接当前网页的 /ws/game。

Submission

提交约束

Dockerfile

压缩包根目录必须包含 Dockerfile。评测机会会以压缩包根目录作为构建上下文来构建镜像并在对局中运行。

代码内容

不再区分语言类型，镜像里可使用任意语言实现，只要容器启动后能按环境变量 TOKEN 和 SERVER 连接游戏服。

安全限制

平台会拒绝非法路径、符号链接、超过 10 MB 的上传文件，以及解压后超过 50 MB 的压缩包。

Endpoints

服务入口

路径 用途
/ 比赛平台首页、登录注册、提交代码和排行榜。
/docs/ 当前文档页。
/api/ 提交系统 API，由 nginx 反代到 FastAPI。
/spectator/ 公开观战台。
/ws/game 观战台 WebSocket，由 nginx 反代到游戏服务器。
