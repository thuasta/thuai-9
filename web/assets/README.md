# 美术资产说明

本目录已包含一组程序化 PNG 占位资产，文件名与前端引用一致。后续拿到正式 AI 美术稿时，直接用同名文件覆盖即可；图片缺失时前端仍会自动降级为纯色背景。

## sidebar-brand.png
- 用途：左侧边栏顶部品牌图
- 建议尺寸：480×160px（2x）
- 图片生成提示词：
  ```
  华清街夜景，俯瞰视角，霓虹灯，黄金交易所招牌，赛博朋克风格，深色调，金色光晕，横幅构图，16:3比例，无文字
  ```

## settlement-character.png
- 用途：每日结算弹窗左侧人物插画
- 建议尺寸：400×600px（2x）
- 图片生成提示词：
  ```
  西装革履的商人大亨，半身像，华清街夜景背景，漫画插画风格，深色背景，金色调，自信表情，无文字，竖版构图
  ```

## bg-texture.png
- 用途：全局背景纹理（body background-image，低透明度叠加）
- 建议尺寸：1920×1080px，可平铺
- 图片生成提示词：
  ```
  深色金融交易所大厅，俯瞰视角，极低饱和度，暗调，适合作为网页背景纹理，无人物，无文字，细节丰富但不抢眼
  ```
