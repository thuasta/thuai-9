# 当前布局说明

当前页面采用固定尺寸的双栏布局，整体是桌面优先设计。

- 整体容器 `app-shell` 分为左右两栏：左侧约 20%，右侧约 80%。
- 左侧是 `sidebar`，包含：
  - 品牌与状态信息
  - 连接与配置控件
  - 视图切换菜单
- 右侧是 `main-shell`，分为两部分：
  - 顶部 `main-header`：显示当前视图标题，以及价格模式、间隔、快报入口等工具
  - 下方 `content-shell`：承载具体视图内容

当前主要有 5 个视图：

1. `main`：主界面，左侧是 K 线与成交量，右侧是盘口信息
2. `logs`：日志视图，显示事件流和每日总结
3. `rankings`：排行视图，显示比分和玩家摘要
4. `info`：信息视图，显示玩家资产、挂单、策略
5. `debug`：调试视图，提供手动下单、撤单、研报、技能操作

补充说明：

- `info` 和 `debug` 是 player 模式专属视图，observer 模式下不显示。
- 页面还包含多个弹窗层（如结算、事件流、璃月快报、详情），用于展示附加信息。
- 当前样式以固定布局为主，`body` 设置了最小宽高，页面不会按移动端方式自适应收缩。
